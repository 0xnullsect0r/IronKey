Features

Architecture

Six Rust crates, one tool

ironkey-app

Entry point. Initialises env_logger and launches the iced application.

ironkey-ui

iced 0.14 PaneGrid application. Manages the three-panel layout, all message routing, modals, and system tick.

ironkey-drives

Parses /sys/block/ to enumerate drives and partitions. Reads SMART data via ATA ioctl. Handles format, clone, and wipe operations.

ironkey-mount

Detects filesystem types and mounts LUKS, NTFS, APFS, BitLocker, and generic volumes.

ironkey-browser

Directory listing, file operations (copy/move/rename/delete), content search, and file viewer.

ironkey-terminal

PTY spawner with VTE ANSI parser and terminal state renderer.

Get Started

From clone to booted USB in 5 steps

1

Install Rust

Install the Rust toolchain via rustup if you don't have it already.

sh 
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
2

Clone the repo

Clone the IronKey repository and enter the project directory.

sh 
git clone https://github.com/0xnullsect0r/IronKey.git && cd IronKey
3

Build the ISO

Run the build script on a Debian or Ubuntu host. Requires root.

sh 
sudo bash os/build.sh ironkey.iso
4

Flash to USB

Write the ISO to your USB drive. Replace /dev/sdX with your device path.

sh 
sudo ./write-usb.sh --drive /dev/sdX
5

Boot

Set USB as first boot device in BIOS/UEFI and reboot. IronKey loads automatically.
