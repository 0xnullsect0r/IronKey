Getting Started

Everything you need to build, flash, and boot IronKey in under 30 minutes.

Prerequisites

Before you begin, make sure your build machine meets these requirements.

  • ✓ Debian 12 or Ubuntu 22.04+ (for the ISO build script)
  • ✓ Root / sudo privileges
  • ✓ At least 4 GB of free disk space in /tmp
  • ✓ Internet connection (debootstrap pulls packages)
  • ✓ Rust stable toolchain (installed in Step 1)
  • ✓ Required packages: debootstrap squashfs-tools grub-pc-bin grub-efi-amd64-bin xorriso mtools
ℹ️
Other Linux distros

Building on other Linux distros is possible but unsupported — debootstrap is available on most Debian-derived systems. On Arch: sudo pacman -S debootstrap.

Step 1: Install Rust

Install the Rust toolchain via rustup:

bash 
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Reload your shell's PATH to activate the newly installed toolchain:

bash 
source "$HOME/.cargo/env"

Verify the installation:

bash 
rustc --version
cargo --version

Step 2: Clone the Repository

Clone the IronKey source code from GitHub and enter the project directory:

bash 
git clone https://github.com/0xnullsect0r/IronKey.git
cd IronKey

Step 3: Build the ISO

Install the required system packages first:

bash 
sudo apt install debootstrap squashfs-tools grub-pc-bin grub-efi-amd64-bin xorriso mtools

Then run the build script to produce the bootable ISO image:

bash 
sudo bash os/build.sh ironkey.iso
ℹ️
Build time

The build takes 5–15 minutes on a fast machine. It bootstraps a full Debian bookworm rootfs, compiles the IronKey binary, and packs everything into a hybrid ISO.

💡
Custom output path

You can pass a custom output path: sudo bash os/build.sh /path/to/my-ironkey.iso

Step 4: Flash to USB

⚠️
Data loss warning

This will erase all data on the target drive. Verify the device path with lsblk before running.

Use the included helper script to identify and write to your USB drive:

bash 
lsblk
sudo ./write-usb.sh --drive /dev/sdX --iso ironkey.iso

Or write manually with dd:

bash 
sudo dd if=ironkey.iso of=/dev/sdX bs=4M status=progress conv=fsync && sync
ℹ️
Other platforms

On Windows, use Rufus (https://rufus.ie). On macOS, use balenaEtcher or dd. See the full installation guide for platform-specific instructions.

Step 5: Boot IronKey

  1. Insert the USB drive into the target machine.
  2. Enter BIOS/UEFI (F2, F12, Del, or Esc at startup).
  3. Disable Secure Boot.
  4. Set the USB drive as the first boot device.
  5. Save and reboot.

IronKey boots silently into the fullscreen GUI in seconds. No login, no desktop manager — just the tool.

You're in!
